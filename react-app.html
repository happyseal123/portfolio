Portfolio Pieces:

Cooking Recipe React App

Context & Objective
  • For this project, I wanted to create an interface through which users can browse and save cooking recipes. My main goal was to become comfortable using JavaScript and React.
  • While I didn't include a consumer component in this site, I purposely wanted to craft a page that could easily be extended into a consumer experience (such as a recipe site from which users can order groceries).
Research
  • Before crafting my own recipe site, I analyzed existing sites for cooking recipes, including New York Times Cooking and Martha Stewart's site:
  • By looking at a variety of recipe sites, I came up with a list of features I wanted to include in my recipe site:
    • Recipe cards displayed in a grid with eye-catching images
    • Filtering functionality (filtering by recipe type and recipe difficulty)
    • Sorting functionality (sorting by length of prep time)
    • The ability to save and remove recipes to refer back to later
Coding with React
  • With a list of features I wanted to include in mind, I set to work coding the site using JavaScript with React. Through trial and error and lots of learning, I ultimately was able to achieve all the functionality I desired.
  • My deployed cooking recipe React application can be found here.
Takeaways
  • It was exciting to sharpen my skills in JavaScript and React. In the future, I am eager to combine those skills with my understanding of design principles to build visually stunning consumer sites with custom functionality.
  • While conducting research by examining other recipe sites was useful, I think out-of-the-box thinking is more valuable than merely copying and/or improving upon features from existing sites.
